Smooth command (Processing menu)

This command smooths all spectra by using the Savitzky-Golay smoothing procedure (see below).

 

 

 

 

 

 

 

 

 

 

On selection of the function, a dialog is activated to allow the specification of the smoothing filter width (see below).

 

 

 

The filter width in the dialog is the value of n in 2n+1 for the entire filter width.

The Savitzky-Golay smoothing procedure is implemented here using a quadratic function.

Smoothing is useful for removing noise and enhances the ability of SpecAlign to pick peaks.

For details of the Savitzky-Golay smoothing procedure see the following reference:

  • Savitzky, A. and Golay, M. (1964) Smoothing and Differentiation of Data by Simplified Least Squares Procedures, Analytical Chemistry, 36(8), 1627-39.
