Denoise command (Processing menu)

This command smooths all spectra by using a Symmlet8 wavelet transform and then applying soft thresholding to the wavelet (see below).

 

 

 

 

 

 

 

 

 

 

On selection of the function, a dialog is activated to allow the specification of the threshold value (see below).

 

 

 

Typically, a threshold of 0.5 is used defined as the median absolute value of the wavelet coefficients. The actual threshold (T) is calculated as:

T = sqrt(2*log(n))*t/sqrt(n)

where n is the length of the input and t is the input threshold from the dialog (see reference).

Denoising is useful for removing noise and enhances the ability of SpecAlign to pick peaks.

Reference:

  • Donoho D. L. (1994) De-Noising By Soft-Thresholding, IEEE Transactions on Information Theory, 41(3), 613-27.
