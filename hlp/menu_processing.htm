Processing menu commands

The  Processing menu offers the following commands:

Normalize TIC Normalize all spectra using the total ion count/signal.
Rescale to positive Removed negative intensities by making the minimum value 0.
Remove negative Coverts all negative values to 0.
Relative intensities Converts all spectra to relative intensity on the y axis.
Subtract baseline Subtracts baseline off all spectra at a specified threshold.
Scale Scale all spectra by a specfied factor.
Smooth Smooths all spectra using the Savitzky-Golay smoothing procedure.
Denoise Denoises all spectra using the a Symmlet8 wavelet and soft thresholding.
Bin spectra Bins all spectra to a specified window.
Align spectra Aligns all spectra to an average spectrum.

Note: Processing commands may be undone by using Undo functions under the Edit menu.
