Export command (File menu)

This command gives options to export the loaded spectra in SpecAlign to three different sources:

Single File:

Exports the currently loaded spectra to a single CSV file in the following format:

m/z Sample 1 Sample 2 Sample 3 Sample 4 Sample 5
...

12

 15 16 18 ...
3453.0 16 20 18 20 ...
3454.0 19 55 45 100 ...
3455.0 48 88 96 201 ...
3456.0 148 200 156 133 ...
3457.0 115 133 110 120 ...
3458.0 20 18 50 80 ...
... 12 5 22 3 ...

Where there the first column consist of the m/z values and the remaining columns to the right consist of the intensity values of each sample. Note that the above is only an example, where the spectrum is a mass spectrum.

Multiple Files :

Gives the option to export each spectrum to an individual file in a specified folder in CSV format.

Average spectrum:

An option to export the average spectrum to a CSV file. See Generate average spectrum for a description of the average spectrum.

Peak Intensity:

An option to export the intensity of all common peaks (as defined by average spectrum) for each spectrum.

Peak Area:

An option to export the area of all common peaks (as defined by average spectrum) for each spectrum based on the modeled peak functions.
