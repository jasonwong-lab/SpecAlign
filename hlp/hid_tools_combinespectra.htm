Combine spectra command (Tools menu)

This command allows the user to view the average spectrum for specified spectra. For example, one might wish to view the average spectrum for only a particular class of spectra. The combined spectra is displayed at the top position of the window (but below the average spectrum) and is plotted in green. The combined spectrum once generated may be manipulated in the same way as any other spectrum.

A dialog enables selection of the spectra that one wishes to combine.
