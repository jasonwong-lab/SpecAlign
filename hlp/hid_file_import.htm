Import command (File menu)

This command gives options to import spectrum files from three different sources:

File:

This allows importing of either a single spectrum list or a file containing multiple spectrum list in CSV or TXT (tab-delimited) format.

For the multiple spectrum list, the format of list must be in the form of:

m/z Sample 1 Sample 2 Sample 3 Sample 4 Sample 5
...

12

 15 16 18 ...
3453.0 16 20 18 20 ...
3454.0 19 55 45 100 ...
3455.0 48 88 96 201 ...
3456.0 148 200 156 133 ...
3457.0 115 133 110 120 ...
3458.0 20 18 50 80 ...
... 12 5 22 3 ...

Where there the first column consist of the m/z values and the remaining columns to the right consist of the intensity values of each sample. Note that the above is only an example, where the spectrum is a mass spectrum. There is no limit on the number of decimal places for m/z or intensity values. There is also no requirement for headings on top of each column.

Note: From version 1.1, SpecAlign supports importing of files with different x-axis values. Data points that do not match the ones already loaded will be estimated by linear interpolation.

Folder:

Gives the option to load all files from within a folder that contains multiple files each consisting of spectra lists in CSV or TXT (tab delimited) format.

Average spectrum:

An option to load the average spectrum from a CSV or TXT (tab delimited) file. See Generate average spectrum for a description of the average spectrum.
