Model peaks command (Spectrum menu)

Use this command to use bring up a dialog (see below) which gives options to modeled the peaks in all spectra based on the selected function using the Levenberg-Marquart algorithm.

Peak modelling not only enables the areas of peaks to be accurately estimated, but it also allows overlapping peaks to be resolves therefore giving more accurate area measurements compared to integration of the area under the peak curve. For more details about peak modelling refer to:

Wong J.W.H., Jessop, D.M., Cagney, G., Cartwright, H.M. Modeling and Resolving Overlapping Peaks for Accurate Quantitation of Mass Spectrometry Biomarker Signals, IMSC2006, Prague.  http://abstracts.imsc2006.org/abstract.php?ID=440

Following modelling, the modelled peaks are displayed in dotted green (see below) and the option will be available to export peak areas.

Once the peaks have been modelled, it is possible to extract peak areas directly through File-> Export -> Peak Area. An option will then be available to select the region of the peak signal desired based on the peak model (below).
