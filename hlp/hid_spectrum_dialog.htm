Spectrum dialog

This dialog is activated by clicking the button on the right hand side of each spectrum (see below). Each spectrum dialog is unique to its adjacent spectrum and changes to settings in the dialog will only be reflected in its corresponding spectrum.

The spectrum dialog provides the following functions (as labeled above):

  1. Moves a spectrum to the specified position in the spectra display. Example, entering 1 for any spectrum will move it to the top of the display.
  2. The name for a spectrum.
  3. The type class for a spectrum.
  4. Any description about a spectrum.
  5. Smoothing filter width. Entering a value will smooth this spectrum only using the Savitzky-Golay procedure.
  6. Subtracts baseline for this spectrum only.
  7. Copies the current spectrum to clipboard as a bitmap.
  8. Selection of spectrum line type as displayed in stacked or overlapping view.
