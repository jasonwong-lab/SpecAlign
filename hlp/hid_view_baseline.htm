Baseline command (View menu)

The view baseline command allows control of the sensitivity for the baseline selected as detailed in the screenshot below:

The baseline picking algorithm implements a restrained moving average, where only values less than the local average intensity are added to the global moving average.

The local moving average is specified as 1/4 of the global moving average. The dialog option allows selection for the size of the global moving average window expressed as a percentage of the full range of the spectrum.
