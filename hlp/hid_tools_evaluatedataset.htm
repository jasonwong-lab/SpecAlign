Evaluate dataset command (Tools menu)

This command allows the user to rapidly evaluate the quality of the dataset by checking for spectra which may be of poor quality relative to others within the dataset.

Two metrics are used to evaluate each spectrum relative to other spectra.

The first is the average Pearson's correlation coefficient of each spectrum against all other spectra within the dataset. For this measurement, users have the choice to compare each spectrum in separate segments. Furthermore, users may also choose the range in which to compare spectra. The correlation coefficient is effective in detecting spectra which are unlike others in the dataset in terms of shape.

The second metric for evaluating spectra quality within the dataset is the normalization factor. This factor is essentially the scaling factor calculated applied during a total ion current normalization process. The normalization factor is effective in detecting spectra that substantially vary in total ion intensity relative to other spectra of the dataset.

When the command is executed, users are first given the choice of an ,/z range to correlate spectra and the number of segments to partition the spectra into for correlation. Note that 1 segment means that the whole spectrum is correlated with others directly. Increasing the number of segments may be useful if a spectrum is dominated by high intensity peaks in one region, with lower intensity peaks exist in others regions of the spectrum.

Following the "Evaluate dataset" dialog, users will then be prompted for a location to save the results of the evaluation.

The results will be output as a CSV file which may be viewed in spreadsheet software such as Microsoft Excel(TM).

The output file is divided into two sections, the first is a list of the two metrics for each spectrum (below).

A2 0.9455 1.0433
A3 0.6544 1.6934

The second section is a matrix of individual correlation coefficients of each spectrum against each other (below).

(Note that the matrix is symmetric)

This matrix can be plotted as a "heatmap" (using external software such as Matlab) to allow for rapid visual identification of spectra that may be different from others.
