File menu commands

The File menu offers the following commands:

New Creates a new document.
Open Opens an existing document.
Close Closes an opened document.
Save Saves an opened document using the same file name.
Save As Saves an opened document to a specified file name.
Import Import spectra or average spectrum file.
Export Exports spectra or average spectrum to file.
Print Prints a document (in stack mode only).
Exit Exits SpecAlign.
