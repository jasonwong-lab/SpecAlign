Spectrum menu commands

The  Spectrum menu offers the following commands:

Pick peaks Picks peaks from all spectra.
Manual pick peak Activates manual pick peak function.
Remove peaks Removes all peaks from all spectra if they have been picked.
Manual remove peaks Manually removes all peaks in a selected region.
Plot peaks Plots picked peaks as bars on each spectrum.
Model peaks Model picked peaks for all spectra.
Track Activates a cursor that shows the <x,y> position of the cursor on a spectrum
Zoom in Activates zoom in function
Crop Activates cropping function
Crop from Activates cropping function allowing manual selection of range
Remove spectrum Removes one spectrum or several spectra from all that is loaded
Generate average spectrum Creates a new average spectrum from loaded spectra.
Merge with average spectrum Merges newly loaded spectra with existing average spectrum.
