Scale command (Processing menu)

This command scales all spectra by a specified factor, either by enhancing or suppressing small peaks (see below).

 

 

 

 

 

 

 

 

 

 

On selection of the function, a dialog is activated to allow the specification of the scaling factor (see below).

 

 

 

 

The scaling factor, f, represents the power term for scaling each data point, x, within each spectrum. i.e. xf, such that a factor of 2 results in squaring all data points and a factor of 0.5 results in square rooting all data points.
