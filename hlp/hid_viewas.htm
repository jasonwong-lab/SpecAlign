View as command (View menu)

Gives three options to view the loaded spectra as:

Line graph (default):

Each spectrum is displayed with its own axis as a line graph (see below).

 

 

 

 

 

 

 

 

 

 

Bar graph:

Same as line graph, but points are displayed as bars rather than joined by lines. Note that when zoomed out, it looks similar to line graphs with the area below the spectrum filled in.

 

Stacked graph:

All spectra are displayed in one axis stacked on top of one another (see below). When printing or copying to clipboard via the edit menu, the image is printed or exported as a stacked graph.

Note that in stacked mode, all commands such as peak picking are still functional, but the result cannot be directly visualized until the viewing mode is switched back to line or bar graph.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Overlapping graph:

All spectra are displayed in one axis overalpping one another (see below). When printing or copying to clipboard via the edit menu, the image is printed or exported as a stacked graph.

Note that in overlap mode, all commands such as peak picking are still functional, but the result cannot be directly visualized until the viewing mode is switched back to line or bar graph.
