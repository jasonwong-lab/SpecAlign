Bin spectra command (Processing menu)

This command bins values of all spectra to a specified bin size (see below).

 

 

 

 

 

 

 

 

 

 

On selection of the function, a dialog is activated to allow the specification of the bin size (see below).

 

 

 

The bin size is specified in the unit of the x-axis. For example, if the x-axis is m/z, then a bin of 3, combines all values into 3 m/z bins starting from the first m/z value (see below).
