Toolbar

The toolbar is displayed across the top of the application window, below the menu bar. The toolbar provides quick mouse access to many tools used in SpecAlign

To hide or display the toolbar, click Toolbar from the View menu.

List of functions (from left to right):

  1. New document
  2. Open new SpecAlign file
  3. Save to file
  4. Import single file
  5. Import from folder
  6. Import average spectrum
  7. Export all to a single file
  8. Export to a folder
  9. Export average spectrum
  10. View as line graph (default)
  11. View as bar graph
  12. View all spectra as stacked graph
  13. View all spectra as overlapping graph
  14. View (pick) baseline
  15. Activate tracking mode
  16. Activate zoom in
  17. Undo zoom in
  18. Revert to full range
  19. Activate cropping tool
  20. Remove spectrum
  21. Pick peaks
  22. Manual pick peak
  23. Remove peaks
  24. Manual remove peaks
  25. Plot picked peaks
  26. Model peaks
  27. Normalize TIC of all spectra
  28. Convert to relative intensity
  29. Subtract baseline
  30. Scale spectra
  31. Smooth spectra
  32. Smooth spectra
  33. Bin spectra
  34. Align spectra
  35. Generate average spectrum
  36. About SpecAlign
  37. Activate "What is" tool
